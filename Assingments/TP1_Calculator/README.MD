Desenvolver, em Flutter, uma calculadora simples com as seguintes características:

contendo apenas as quatro operações aritméticas básicas (soma, subtração, multiplicação e divisão);
exibir a operação no campo de texto correspondente à medida que o usuário digita os valores (ou usar o recurso de colar da área de transferência, se você quiser, mas isso pode trazer complicações);
exibir o resultado da operação ou uma mensagem de erro quando o botão de calcular for pressionado ou já ir calculando à medida que uma operação formada por dois operandos e um operador entre eles estiver completada;
inclua na documentação (comentários), uma explicação sobre o formato de operação válida;
lembre-se de incluir também o tratamento de erro por divisão por zero.
Documente em comentários as linhas de comando, classes, funções etc. que julgar pertinente para que seu código torne-se bastante didático. Essa documentação também será avaliada.

Você pode usar trechos de código apresentados como exemplo no material de apoio ou códigos provenientes de outras fontes. O importante é que você saiba explicar todo o seu código. "Steal like an artist. (Austin Kleon)"
